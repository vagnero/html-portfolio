Página de Cadastro (CLIENTE):

Descrição:

Esta página foi criada consumindo a api ViaCep, permitindo pegar o endereço dos correios por meio do cep inserido, em tempo real. No formulário é necessário inserir nome, email, senha, cep, etc...

Página de login (CLIENTE):

Descrição:

Na página de login, é buscado no banco de dados primeiramente o email, caso não exista, então é informado ao usuário, se existir em seguida ele verifica a senha, caso ela esteja incorreta, então será informado ao usuário.

Página de compras:

Descrição:

A página de compras, quando o usuário está logado, é liberada, exibindo todos os produtos comprados pelo usuário, além disso vocês podem ver um carrinho de compras e a foto de perfil do usuário, que pode ser trocada a qualquer momento.

Página de perfil:

Descrição:

A página de perfil permite alterar o endereço ou a foto de perfil.

Página de apagar clientes (ADMIN):

Descrição:

Nesta página, o admin pode apagar clientes selecionados. Há também a tela de login e de cadastro, ela é similar ao de cliente, mas com menos campos então não será exibida.

Página de crud produtos (ADMIN):

Descrição:

Nesta página, o admin pode visualizar, apagar e editar produtos selecionados. podendo colocar o nome, foto, quantidade, preço, etc...

Página de editar produtos (ADMIN):

Descrição:

Nesta página, o admin bota em prática a edição dos produtos.
